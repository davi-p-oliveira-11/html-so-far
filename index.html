O que é o HTML

(Voltar)

HTML (HyperText Markup Language) é a linguagem que usamos para criar a estrutura de páginas na internet. Com ele, podemos definir coisas como títulos, parágrafos, imagens e links, organizando o conteúdo para que os navegadores mostrem tudo de forma correta. Não é uma linguagem de programação, mas é fundamental para qualquer site, já que, junto com o CSS e o JavaScript, permite criar páginas interativas e bonitas. Basicamente, o HTML é o esqueleto de qualquer site na web.

Estrutura básica do HTML

(Voltar)

Imagine que voce quer construir uma casa. Voce precisa de uma estrutura sólida para que a casa seja bem delimitada e não caia.

De forma Semelhante a estrutura básica do HTML funciona assim.

A primeira parte é a declaração do tipo de documento, que informa ao navegador qual versão do HTML está sendo utilizada. Em seguida, temos a tag html, que engloba todo o conteúdo da página. Dentro dela, encontramos duas seções principais: o head e o body. O head é como a identidade da página, onde colocamos o título, metadados e links para arquivos externos. Já o body é o corpo da página, onde inserimos todo o conteúdo visível ao usuário, como textos, imagens, vídeos e outros elementos.

Para entender melhor, pense no head como um documento de identidade e no body como o corpo de um livro. O documento de identidade contém informações como o nome, data de nascimento e outras particularidades da pessoa, enquanto o corpo do livro contém a história que você quer contar. No HTML, o head contém informações sobre a página, como o título que aparece na aba do navegador, enquanto o body contém o conteúdo que será exibido na tela.

O que são tags em HTML

(Voltar)

Tags HTML são como as palavras-chave que utilizamos para estruturar e dar significado ao conteúdo de uma página web. Elas são como blocos de construção que, quando combinados, formam a estrutura completa de uma página. Cada tag possui um papel específico e define como o navegador deve interpretar e exibir o conteúdo.

Imagine que você está escrevendo um livro. As tags HTML seriam como as diferentes formatações que você utilizaria: negrito para destacar palavras importantes, itálico para enfatizar frases, parágrafos para organizar as ideias, etc. No HTML, as tags são delimitadas por colchetes angulares (< e >) e geralmente vêm em pares, com uma tag de abertura e uma de fechamento. Por exemplo, a tag p define um parágrafo, e a tag p indica o fim do parágrafo.

Atributos das tags

(Voltar)

Atributos são como adjetivos que fornecem informações adicionais sobre as tags HTML. Eles são inseridos dentro da tag de abertura e modificam o comportamento ou a aparência do elemento. Imagine que uma tag é um substantivo e os atributos são adjetivos que descrevem esse substantivo de forma mais precisa. Por exemplo, na tag img, o atributo src indica o endereço da imagem que será exibida, e o atributo alt fornece um texto alternativo para usuários com deficiência visual ou caso a imagem não carregue. Os atributos permitem que você personalize e refine a estrutura e o conteúdo da sua página web

Textos no HTML

(Voltar)

Textos são a essência de qualquer página web. Em HTML, utilizamos tags específicas para estruturar e formatar o texto da forma que desejamos. As tags mais comuns são: p para parágrafos, h1 a h6 para títulos de diferentes níveis, strong para texto em negrito e em para itálico. Além disso, podemos utilizar atributos como align para alinhar o texto e style para aplicar estilos CSS. Com essas ferramentas, podemos criar textos claros, organizados e visualmente atraentes em nossas páginas web.

Listas

(Voltar)

Listas em HTML: Organizando o Conteúdo Listas são elementos essenciais para apresentar informações de forma clara e organizada em uma página web. O HTML oferece dois tipos principais de listas: ordenadas e não ordenadas. As listas ordenadas ol são utilizadas quando a ordem dos itens é importante, como em uma lista de passos ou um ranking. Cada item da lista é numerado automaticamente. Já as listas não ordenadas ul são usadas quando a ordem dos itens não é relevante, como em uma lista de ingredientes ou características. Cada item é precedido por um marcador, como um ponto ou um círculo. Ao combinar listas com outras tags e atributos, podemos criar estruturas complexas e personalizadas para organizar o conteúdo de nossas páginas web.

Links

(Voltar)

Links, ou hiperlinks, são os elementos que conectam as páginas da web, formando uma vasta rede de informações. Eles funcionam como portas que levam o usuário de um lugar para outro na internet. Em HTML, os links são criados utilizando a tag a, que define uma âncora. Dentro dessa tag, o atributo href indica o endereço (URL) para onde o link irá direcionar. Além de conectar páginas, os links podem ser utilizados para baixar arquivos, enviar e-mails e até mesmo criar marcadores dentro da mesma página. Os links são essenciais para a navegação e interação do usuário com um site, permitindo que ele explore diferentes conteúdos de forma rápida e fácil. A versatilidade dos links torna-os uma ferramenta fundamental para a construção de sites dinâmicos e informativos.
